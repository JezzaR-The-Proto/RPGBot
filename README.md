# RPGBot
 A Discord Bot that simulates a simple RPG.
## Installation
**REQUIREMENTS:**
- Python 3.7 or later.
- Python Modules:
	- discord.py, requests, logging, PIL(Pillow), asyncio.
- At least 10MB of storage.
## Setup
- Clone the repo
- Install required python modules.
- Put your Bot Token into main.py at the end.
- Run main.py
